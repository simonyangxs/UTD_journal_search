## 📖 使用说明

### 🚀 快速开始
1. **勾选期刊**: 在期刊列表中勾选您要搜索的期刊（支持多选）
2. **查看选择**: 已选择的期刊会显示在"已选择期刊"文本框中
3. **输入搜索关键词**: 在文本框中输入您的搜索关键词
4. **选择搜索字段**: 选择在标题、摘要还是全文中搜索
5. **设置时间范围**: 如果期刊支持，可以设置起始和结束年份
6. **开始搜索**: 点击搜索按钮，系统会智能合并同网站期刊并打开搜索页面

### ✅ 期刊选择功能

**复选框选择**:
- 期刊按领域分组显示（运营管理+信息系统、经济学、金融+会计、管理学）
- 直接勾选复选框即可选择期刊
- 支持跨领域多选

**便捷操作**:
- **全选按钮**: 一键选择所有期刊
- **清空按钮**: 一键清空所有选择
- **实时显示**: 已选择期刊实时显示在选择框中
